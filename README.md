# CENG 467 - Natural Language Understanding and Generation Midterm

This repository contains the implementations for the Take-Home Midterm.

## Prerequisites
* **Python version:** `3.11.9`

## Environment Setup
To ensure full reproducibility, please create a virtual environment and install the required dependencies:


# Create and activate a virtual environment
py -3.11 -m venv .venv
# macOS / Linux:
# source .venv/bin/activate
 
# Windows (PowerShell):
# .venv\Scripts\Activate.ps1
#
# Windows (CMD):
# .venv\Scripts\activate.bat

# Install dependencies
pip install -r requirements.txt


## Running the Code
* **Question 1 (Representation Learning):** `python Q1_Classification/q1_classification.py`
* **Question 2 (Named Entity Recognition):** `python Q2_NER/q2_ner.py`
* **Question 3 (Text Summarization):** `python Q3_Summarization/q3_summarization.py`
* **Question 4 (Machine Translation):** `cd Q4_Translation && python visualize.py`
* **Question 5 (Language Modeling):** `python Q5_Language_Modeling/q5_language_modeling.py`
